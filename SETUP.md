"## ?? Step 1: Download/Extract Project" 
"" 
"### If you have a ZIP file:" 
"1. Extract the `FitZone.zip` file to your Desktop" 
"2. Navigate to the extracted folder" 
"" 
"### Using Git:" 
"```bash" 
"git clone https://github.com/yourusername/fitzone.git" 
"cd fitzone" 
"```"

"## ?? Step 2: Open Terminal/Command Prompt" 
"" 
"### Windows:" 
"- Press `Win + R`" 
"- Type `cmd`" 
"- Press Enter" 
"" 
"### Mac/Linux:" 
"- Open Terminal from Applications" 
"" 
"---" 
"" 
"## ?? Step 3: Navigate to Project Folder" 
"" 
"```bash" 
"cd Desktop/FitZone" 
"```" 
"" 
"---" 
"" 
"## ?? Step 4: Create Virtual Environment" 
"" 
"```bash" 
"# Windows" 
"python -m venv venv" 
"" 
"# Mac/Linux" 
"python3 -m venv venv" 
"```" 
"" 
"---" 
"" 
"## ?? Step 5: Activate Virtual Environment" 
"" 
"```bash" 
"# Windows" 
"venv\Scripts\activate" 
"" 
"# Mac/Linux" 
"source venv/bin/activate" 
"```" 
"" 
"**You should see `(venv)` at the beginning of your terminal line.**" 
"" 
"---" 
"" 
"## ?? Step 6: Install Required Packages" 
"" 
"```bash" 
"pip install -r requirements.txt" 
"```"

"## ?? Step 7: Configure Email (For Password Reset & Account Notifications)" 
"" 
"### Recommended setup using environment variables:" 
"" 
"Create or update the `.env` file in the project root with:" 
"" 
"```bash" 
"EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend" 
"EMAIL_HOST=smtp.gmail.com" 
"EMAIL_PORT=587" 
"EMAIL_USE_TLS=true" 
"EMAIL_USE_SSL=false" 
"EMAIL_HOST_USER=fitzone.alerts@gmail.com" 
"EMAIL_HOST_PASSWORD=your_app_password_here" 
"DEFAULT_FROM_EMAIL=\"FitZone <fitzone.alerts@gmail.com>\"" 
"```" 
"" 
"If you deploy on Render with Brevo SMTP, add the Brevo credentials in Render Environment instead of committing them to source control:\n" 
"- EMAIL_HOST=smtp-relay.brevo.com\n" 
"- EMAIL_PORT=587\n" 
"- EMAIL_USE_TLS=True\n" 
"- EMAIL_HOST_USER=<your-brevo-login>\n" 
"- EMAIL_HOST_PASSWORD=<your-brevo-password>\n" 
"- BREVO_API_KEY=<your-brevo-api-key>\n" 
"- BREVO_SENDER_EMAIL=<your-brevo-sender-email>\n" 
"- DEFAULT_FROM_EMAIL=\"FitZone <fitzone.alerts@gmail.com>\"\n" 
"" 
"4. Do not commit `.env` to Git:" 
"" 
"```bash" 
"echo \".env\" >> .gitignore" 
"```" 
"" 
"5. Restart Django after updating `.env`." 
"" 
"### How to get App Password:" 
"1. Go to https://myaccount.google.com/apppasswords" 
"2. Enable 2-Step Verification" 
"3. Select App: Mail, Device: Other" 
"4. Copy the 16-character password" 
""

""

""
